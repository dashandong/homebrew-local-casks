# Dashandong Local-casks

## How do I install these formulae?

`brew install dashandong/local-casks/<formula>`

Or `brew tap dashandong/local-casks` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "dashandong/local-casks"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
